# 2026-06-02 의사결정 드릴다운 요약 배지 추가

## 배경
BUY 차단 카드에서 `의사결정` 화면으로 드릴다운한 뒤에도, 현재 필터가 몇 건을 가리키는지 상단에서 즉시 읽기 어려웠다.

표 하단의 `총 N건`은 존재하지만, 드릴다운 맥락에서는 상단에서 결과 건수를 먼저 보여주는 편이 더 자연스럽다.

## 목표
- 드릴다운 필터가 적용된 상태에서 상단에 결과 요약 배지를 노출
- 운영자가 현재 필터의 규모와 페이지 상태를 바로 파악할 수 있게 한다

## 구현
- `admin_ui/src/components/DecisionsView.tsx`
  - 드릴다운 필터 배너 아래에 summary badge 3개 추가
    - `필터 결과 N건`
    - `현재 페이지 M건`
    - `페이지 P/T`

## 테스트
- `admin_ui/src/__tests__/decisions.test.tsx`
  - 드릴다운 상태에서
    - `필터 결과 1건`
    - `현재 페이지 1건`
    - `페이지 1/1`
  - 노출 확인

## 기대 효과
- 드릴다운 화면에서 현재 필터 범위 파악이 빨라짐
- 표 하단 pagination footer까지 내려가지 않아도 결과 건수를 즉시 확인 가능
