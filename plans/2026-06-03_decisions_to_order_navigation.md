# 2026-06-03 의사결정 화면 → 주문 화면 직접 이동

## 배경
`오늘 BUY 차단` 카드에서 `주문 생성 보기`로 내려가면 BUY 의사결정 목록까지는 볼 수 있었지만, 실제 주문 상세로 가려면 다시 주문 화면에서 수동 검색해야 했다.

운영자는 의사결정과 주문을 한 번에 이어서 확인할 수 있어야 한다.

## 목표
- 의사결정 목록 표에서 주문 생성된 row를 바로 주문 상세로 이동 가능하게 한다
- 의사결정 상세 패널에서도
  - 주문 상세
  - 제출 이력
  로 직접 이동 가능하게 한다

## 구현
- `admin_ui/src/components/DecisionsView.tsx`
  - 표 컬럼 `주문` 추가
    - `order_request_id`가 있으면 `주문 상세 →` 링크 표시
  - 상세 패널의 `주문 정보` 카드 하단에 링크 추가
    - `주문 상세 보기 →`
    - `제출 이력 보기 →`

## 테스트
- `admin_ui/src/__tests__/test-utils/fixtures.ts`
  - 첫 번째 mock decision에 `order_request_id`, `order_status` 부여
- `admin_ui/src/__tests__/decisions.test.tsx`
  - 표의 `주문 상세 →` 링크 href 검증
  - 상세 패널의 `주문 상세 보기 →`, `제출 이력 보기 →` href 검증

## 기대 효과
- 드릴다운 흐름이
  - BUY 차단/주문 생성 카드
  - 의사결정 목록
  - 주문 상세 / 제출 이력
  로 자연스럽게 이어짐
- 주문 화면 재검색 비용 제거
