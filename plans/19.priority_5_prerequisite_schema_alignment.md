## Step 2: _dataclass_to_dict() 재귀 변환

[`decision_orchestrator.py:486`](../src/agent_trading/services/decision_orchestrator.py:486)의 `_dataclass_to_dict()`가 nested dataclass를 재귀적으로 직렬화하도록 수정.

```python
def _dataclass_to_dict(obj: object) -> dict[str, object]:
    """Recursively convert a frozen dataclass to a JSON-compatible dict."""
    if not hasattr(obj, "__dataclass_fields__"):
        return {}
    result: dict[str, object] = {}
    for field_name in obj.__dataclass_fields__:
        value = getattr(obj, field_name)
        if isinstance(value, UUID):
            result[field_name] = str(value)
        elif hasattr(value, "__dataclass_fields__"):
            result[field_name] = _dataclass_to_dict(value)
        elif isinstance(value, tuple):
            result[field_name] = tuple(
                _dataclass_to_dict(v) if hasattr(v, "__dataclass_fields__")
                else str(v) if isinstance(v, UUID)
                else v
                for v in value
            )
        else:
            result[field_name] = value
    return result
```
