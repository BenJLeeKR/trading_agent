# Priority 5-1: Event Interpretation Agent — DeepSeek Provider 연결

## 목적

Event Interpretation Agent 1개를 실제 DeepSeek API에 연결한다. Agent는 항상 `AIProviderClient` protocol을 통해 Provider와 통신하며, orchestrator는 Provider SDK를 직접 호출하지 않는다.

---

## 1. DeepSeek 연동 방식

### 1.1 HTTP REST 기반 클라이언트 (SDK 미사용)

DeepSeek은 OpenAI-compatible API를 제공한다. 즉, OpenAI SDK 없이도 HTTP REST로 동일한 endpoint를 호출할 수 있다.

| 항목 | 값 |
|------|-----|
| Base URL | `https://api.deepseek.com` |
| Chat endpoint | `POST /v1/chat/completions` |
| 인증 | `Authorization: Bearer <api_key>` 헤더 |
| JSON mode | `response_format={"type": "json_object"}` 지원 |
| 모델 | `deepseek-chat` |

**장점**: OpenAI-compatible이므로 추후 OpenAI로 전환 시 URL/Key만 변경하면 됨.

### 1.2 Structured output 강제 방식

```
Agent → system prompt (JSON schema 설명)
     → user prompt (context, events)
     → DeepSeek API (response_format=json_object)
     → raw JSON string
     → parse → validate → EventInterpretationOutput
```

1. System prompt에 `EventInterpretationOutput` JSON schema를 포함
2. `response_format={"type": "json_object"}` 로 JSON 출력 강제
3. 응답 JSON을 `EventInterpretationOutput`으로 역직렬화 (`_dataclass_to_dict` 역방향)
4. 파싱 실패 시 → log warning → default `EventInterpretationOutput()` 반환 (safe fallback)

### 1.3 필요한 의존성

`httpx`는 이미 프로젝트에서 사용 중(`opendart_adapter.py`, `rest_client.py`). `pyproject.toml`에 명시적으로 추가한다.
다른 SDK 의존성은 추가하지 않는다.

### 3.2 Provider Client — [`provider_client.py`](src/agent_trading/services/ai_agents/provider_client.py)

```python
class OpenAICompatibleClient:
    """HTTP-based OpenAI-compatible provider client.
    
    Implements ``AIProviderClient`` protocol.
    Works with DeepSeek, OpenAI, Ollama, or any OpenAI-compatible endpoint.
    """

    def __init__(
        self,
        *,
        api_key: str,
        base_url: str = "https://api.deepseek.com",
        timeout_seconds: int = 30,
    ) -> None:
        self._api_key = api_key
        self._base_url = base_url.rstrip("/")
        self._timeout = timeout_seconds
        self._client: httpx.AsyncClient | None = None

    async def _get_client(self) -> httpx.AsyncClient:
        if self._client is None:
            self._client = httpx.AsyncClient(
                base_url=self._base_url,
                timeout=httpx.Timeout(self._timeout),
                headers={"Authorization": f"Bearer {self._api_key}"},
            )
        return self._client

    async def generate_structured(
        self,
        *,
        model_id: str,
        system_prompt: str,
        user_prompt: str,
        response_format: type,
        temperature: float = 0.0,
        seed: int | None = None,
    ) -> object:
        """Send a chat completion request and parse JSON response."""
        client = await self._get_client()
        
        body: dict[str, object] = {
            "model": model_id,
            "messages": [
                {"role": "system", "content": system_prompt},
                {"role": "user", "content": user_prompt},
            ],
            "temperature": temperature,
            "response_format": {"type": "json_object"},
        }
        if seed is not None:
            body["seed"] = seed

        response = await client.post("/v1/chat/completions", json=body)
        response.raise_for_status()
        data = response.json()
        raw_content: str = data["choices"][0]["message"]["content"]

        # Parse JSON into the target dataclass
        parsed = json.loads(raw_content)
        return response_format(**parsed)  # dataclass constructor
```

**에러 처리**:
- HTTP 오류 → raise (`generate_structured` 호출자(caller)인 agent가 catch)
- JSON 파싱 실패 → raise (agent가 catch → default fallback)
- dataclass validation 실패 → raise (agent가 catch → default fallback)

**raw_output**: agent가 `generate_structured`의 raw content를 recorder에 전달하는 구조.

### 3.3 Real EventInterpretationAgent — [`event_interpretation.py`](src/agent_trading/services/ai_agents/event_interpretation.py)

`StubEventInterpretationAgent` 유지 + `EventInterpretationAgent` (real) 신규 추가.

```python
class EventInterpretationAgent:
    """Real Event Interpretation Agent — calls DeepSeek via AIProviderClient."""

    def __init__(
        self,
        provider_client: AIProviderClient,
        *,
        model_id: str = "deepseek-chat",
        schema_version: str = "v1",
    ) -> None:
        self._provider = provider_client
        self._model_id = model_id
        self._schema_version = schema_version

    @property
    def agent_name(self) -> str:
        return "event_interpretation"

    @property
    def schema_version(self) -> str:
        return self._schema_version

    async def run(
        self, request: AgentExecutionRequest
    ) -> EventInterpretationOutput:
        try:
            system_prompt = self._build_system_prompt()
            user_prompt = self._build_user_prompt(request)
            
            raw_output_obj = await self._provider.generate_structured(
                model_id=self._model_id,
                system_prompt=system_prompt,
                user_prompt=user_prompt,
                response_format=EventInterpretationOutput,
            )
            
            # raw_output_obj is EventInterpretationOutput
            result: EventInterpretationOutput = raw_output_obj
            
            # Override metadata fields from request
            result = EventInterpretationOutput(
                schema_version=result.schema_version or self._schema_version,
                agent_name=result.agent_name or self.agent_name,
                decision_context_id=result.decision_context_id or (
                    str(request.decision_context_id) if request.decision_context_id else None
                ),
                symbol=result.symbol or request.context.symbol,  # if available
                issuer_code=result.issuer_code,
                events=result.events,
                aggregate_view=result.aggregate_view,
            )
            return result
            
        except Exception:
            logger.warning(
                "EventInterpretationAgent failed — returning default output "
                "(safe fallback). decision_context_id=%s",
                request.decision_context_id,
                exc_info=True,
            )
            return EventInterpretationOutput()

    def _build_system_prompt(self) -> str:
        return (
            "You are an Event Interpretation Agent for a trading system. "
            "Analyze the following external events and produce a structured "
            "interpretation output.\n\n"
            "Output must be valid JSON matching this schema:\n"
            f"{json.dumps(_generate_json_schema(EventInterpretationOutput), indent=2)}"
        )

    def _build_user_prompt(self, request: AgentExecutionRequest) -> str:
        context = request.context
        events = context.recent_events or []
        return (
            f"Symbol: {context.score.symbol if context.score else 'N/A'}\n"
            f"Recent events ({len(events)}):\n"
            + "\n".join(
                f"  - [{e.event_type}] {e.summary}" for e in events[:20]
            )
        )
```

**raw_output 연동**: Agent가 `generate_structured()`의 raw JSON 문자열을 recorder에 전달하도록 수정. `AIProviderClient.generate_structured()`가 `(parsed_object, raw_string)` tuple을 반환하도록 변경하거나, agent에서 raw response를 별도로 확보.

선택: `generate_structured`가 `RawProviderResponse` dataclass를 반환:
```python
@dataclass
class RawProviderResponse:
    parsed: object        # EventInterpretationOutput
    raw_content: str      # raw JSON string from provider
```
